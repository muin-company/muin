## 1. Mac Studio 비용

### 하드웨어 (예상)
- **Mac Studio M4 Ultra**: ~1,000만원
  - 192GB 통합 메모리 (70B 모델 실행 가능)
  - Apple Silicon 최적화

### 실행 가능한 로컬 모델
| 모델 | 파라미터 | 성능 (vs Opus) |
|------|---------|----------------|
| Llama 3.3 70B | 70B | ~60-70% |
| Qwen 2.5 72B | 72B | ~65-75% |
| Mixtral 8x22B | 141B | ~55-65% |
| DeepSeek V3 | 671B (MoE) | ~70-80% |

### 한계
- **성능 격차**: 복잡한 추론, 긴 컨텍스트, 한국어 처리에서 Opus 대비 열세
- **유지보수**: 모델 업데이트, 환경 관리 필요
- **전기 비용**: 고부하시 월 5-10만원 추가

## 2. Claude API 비용

### 가격표 (2026년 기준)

| 모델 | Input | Output |
|------|-------|--------|
| **Opus 4.5** | $5/MTok | $25/MTok |
| Sonnet 4.5 | $3/MTok | $15/MTok |
| Haiku 4.5 | $1/MTok | $5/MTok |

### 예상 사용량 시나리오

**시나리오 A: 가벼운 사용 (하루 2-3시간)**
- Input: 30K tokens/일
- Output: 50K tokens/일
- 일 비용: $0.15 + $1.25 = **$1.40/일**
- 월 비용: **~$42 (약 5.5만원)**

**시나리오 B: 활발한 사용 (하루 8시간+)**
- Input: 100K tokens/일
- Output: 200K tokens/일
- 일 비용: $0.50 + $5.00 = **$5.50/일**
- 월 비용: **~$165 (약 22만원)**

**시나리오 C: 24/7 자율 운영**
- Input: 500K tokens/일
- Output: 1M tokens/일
- 일 비용: $2.50 + $25.00 = **$27.50/일**
- 월 비용: **~$825 (약 110만원)**

## 4. 정성적 비교

### Mac Studio 장점
- ✅ 고정 비용 (예측 가능)
- ✅ 데이터 프라이버시 (로컬 처리)
- ✅ 오프라인 작동
- ✅ 장기적으로 저렴해질 수 있음

### Mac Studio 단점
- ❌ 성능 격차 (Opus 대비 30-40% 낮음)
- ❌ 생산성 저하 → 기회비용
- ❌ 모델 업데이트 수동
- ❌ 초기 자본 필요

### API 장점
- ✅ 최고 성능 (Opus 4.5)
- ✅ 초기 비용 없음
- ✅ 자동 업데이트 (최신 모델)
- ✅ 확장 용이
- ✅ 유지보수 불필요

### API 단점
- ❌ 사용량 비례 비용 (예측 어려움)
- ❌ 인터넷 의존
- ❌ 데이터가 외부로 전송

## 5. 권장사항

### 현재 단계 (Day 0-90): **API 유지**

**이유:**
1. 사업 검증이 우선 - 하드웨어 투자는 검증 후
2. 최고 성능 필요 - 초기에는 품질이 중요
3. 현금 보존 - 1,000만원은 다른 곳에 투자 가능
4. 유연성 - 모델/사용량 조절 가능

### 향후 고려 (Day 90+)

**Mac Studio 구매 고려 시점:**
- 월 API 비용이 50만원+ 지속될 때
- 특정 작업이 반복적이고 로컬 모델로 충분할 때
- 데이터 프라이버시가 중요해질 때

### 하이브리드 전략 (권장)

```
복잡한 작업 (추론, 창작) → Claude Opus API
단순한 작업 (요약, 분류) → 로컬 Haiku 또는 작은 모델
```

## 6. 결론

**지금은 Mac Studio 구매 보류 권장**

- 1,000만원으로 API 기준 **4-7년치** 사용 가능
- 그 사이 로컬 모델 성능도 향상될 것
- 사업이 검증되고 사용량이 확실해지면 재검토

**대안:**
- 월 API 비용 모니터링
- 50만원/월 초과시 Mac Studio 재검토
- 또는 클라우드 GPU (Lambda, RunPod) 고려
