### 💬 댓글 응답 가이드라인

#### **응답 우선순위**

1. **높음 (30분 내 응답)**:
   - 질문 ("이거 어떻게 써요?")
   - 문제 제기 ("버그 발견했어요")
   - 감정적 반응 ("감동했어요", "실망했어요")

2. **중간 (2시간 내 응답)**:
   - 일반 의견 ("좋네요", "기대돼요")
   - 후기 공유 ("이렇게 써봤어요")

3. **낮음 (24시간 내 응답)**:
   - 단순 반응 ("👍", "ㅋㅋㅋ")
   - 봇 댓글

#### **응답 톤 & 스타일**

**검시AI (친근하고 따뜻하게)**:
```
Q: "진짜 무료인가요?"
A: 네! 완전 무료로 시작하실 수 있어요 😊
   나중에 프리미엄 기능이 추가될 예정이지만,
   기본 기능은 영원히 무료입니다!
   지금 바로 시작해보세요 💙
```

**CLI Tools (기술적이지만 친절하게)**:
```
Q: "npm install 에러 나요"
A: Oh no! Can you share the error message?
   Common fixes:
   1. Clear npm cache: npm cache clean --force
   2. Update npm: npm install -g npm@latest
   3. Check Node version: node -v (need v16+)
   
   DM me if still stuck! 🛠️
```

**MUIN (전문적이지만 인간적으로)**:
```
Q: "투명하다고 하는데 수익 모델은?"
A: 좋은 질문입니다!
   
   현재: 무료 제공 (자체 투자)
   향후: 프리미엄 + 기업용 B2B
   
   자세한 내용은 블로그에 포스팅했어요:
   [링크]
   
   더 궁금하신 점 있으면 편하게 물어보세요 🤝
```

### C. 용어집

- **Thread**: 연결된 여러 트윗 시리즈
- **Engagement Rate**: (좋아요+RT+댓글) / 임프레션
- **Impression**: 트윗이 사용자 타임라인에 표시된 횟수
- **RT (Retweet)**: 다른 사람의 트윗을 내 팔로워에게 공유
- **Quote Tweet**: RT에 내 코멘트 추가
- **Mention**: @username으로 특정 사용자 언급
- **CTA (Call To Action)**: 사용자에게 요청하는 행동 (클릭, 가입 등)
