#### 1. "무료면 나중에 유료 전환되겠네요?"
```
정직하게 말씀드리면, 네 맞습니다.

베타 기간:
- 완전 무료 (광고 없음)
- 베타 테스터는 정식 출시 후 6개월 무료

정식 출시 후 예상 가격:
- 무료 플랜 (기본 기능)
- 프리미엄 ₩9,900/월 (전체 기능)

학원비(₩30만/월) 대비 1/30 가격이에요.
```

#### 2. "AI가 틀린 답을 주면 어떡하죠?"
```
중요한 지적입니다!

현재 정확도:
- 개념 설명: ~95%
- 문제 풀이: ~90%

개선 방안:
- 피드백 "이 답변이 도움이 되었나요?"
- 사용자 신고 기능
- 정기 모델 업데이트

완벽하진 않지만, 계속 나아지고 있습니다.
베타 테스트로 더 정확해질 거예요! 🎯
```

#### 3. "경쟁사 [다른 서비스] 랑 뭐가 다른가요?"
```
좋은 질문입니다!

[경쟁사] vs 검시AI:

[경쟁사]:
- 수능 중심
- 비싼 가격 (₩XX만/월)
- 일방향 강의

검시AI:
- 검정고시 전문
- 파괴적 가격 (₩9,900/월)
- 양방향 대화형 AI

타겟이 다르기 때문에 직접 비교는 어렵지만,
검정고시 독학생에게는 저희가 훨씬 적합합니다 😊
```

### ⚠️ Negative Comments (방어적이지 않게)

#### 1. "또 AI 교육 스타트업이네... 실패할 듯"
```
우려 이해합니다.

AI 교육 시장 과포화 맞아요.
하지만 저희는 다릅니다:

1. **틈새 시장**: 검정고시 (블루오션)
2. **가격 파괴**: 기존 대비 1/40
3. **투명성**: #buildinpublic

실패할 수도 있어요. 하지만 해볼 가치는 있습니다.
지켜봐주세요! 😊
```

#### 2. "개인정보 팔아먹으려고 무료인 거 아니에요?"
```
절대 아닙니다.

무료인 이유:
1. 베타 테스트 기간
2. 피드백 수집 목적
3. 제품-시장 적합성 검증

개인정보 판매? **절대 안 합니다.**
- Privacy Policy 명시
- GDPR/CCPA 준수
- 투명한 데이터 정책

의심스러우면 안 쓰셔도 됩니다.
저희는 투명하게 운영합니다 🔐
```

#### 3. "AI가 교육을 망친다"
```
다른 관점에서 보면:

AI는 도구입니다.
- 좋은 의도로 쓰면 → 학습 효율 UP
- 나쁜 의도로 쓰면 → 의존성 UP

검시AI는:
- 공부를 대신해주는 게 아니라
- 학습 여정을 돕는 도구

학원 못 가는 학생들에게
평등한 교육 기회를 주는 것.
그게 저희 철학입니다 📚
```
