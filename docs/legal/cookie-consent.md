# 쿠키 동의 배너 문구 (Cookie Consent Banner Copy)

## 배너 변형 (Banner Variants)

### 변형 1: 간결형 (Concise)

**한국어:**
```
이 웹사이트는 쿠키를 사용하여 사용자 경험을 개선합니다. 
계속 이용하시면 쿠키 사용에 동의하는 것으로 간주됩니다.

[모두 허용] [설정] [거부]
```

**영어:**
```
We use cookies to improve your experience. 
By continuing, you agree to our cookie policy.

[Accept All] [Settings] [Reject]
```

---

### 변형 2: 상세형 (Detailed)

**한국어:**
```
🍪 쿠키 및 개인정보 보호

MUIN은 필수 기능 제공, 서비스 개선, 맞춤형 콘텐츠 제공을 위해 쿠키를 사용합니다.
'모두 허용'을 클릭하면 모든 쿠키에 동의하게 됩니다. 
'설정'에서 원하는 쿠키만 선택할 수 있습니다.

자세한 내용은 [개인정보 처리방침]과 [쿠키 정책]을 참조하세요.

[모두 허용] [필수만] [설정]
```

**영어:**
```
🍪 Cookies & Privacy

MUIN uses cookies to provide essential functions, improve our service, 
and deliver personalized content. By clicking 'Accept All', you consent 
to all cookies. You can choose specific cookies in 'Settings'.

Learn more in our [Privacy Policy] and [Cookie Policy].

[Accept All] [Essential Only] [Settings]
```

## Compliance Checklist for Cookie Consent

### 필수 요소 (Essential Elements)

- [ ] **명확한 정보 제공**: 어떤 쿠키를 사용하는지, 왜 사용하는지 명시
- [ ] **사전 동의 (Opt-in)**: 쿠키 사용 전 명시적 동의 받기
- [ ] **거부 옵션**: 동의 거부 또는 일부만 선택할 수 있는 옵션
- [ ] **쉬운 철회**: 언제든지 동의를 철회할 수 있는 방법 제공
- [ ] **필수/선택 구분**: 필수 쿠키와 선택적 쿠키 명확히 구분
- [ ] **제3자 쿠키 고지**: 제3자(Google, Facebook 등)의 쿠키 사용 시 명시
- [ ] **저장 기간 표시**: 각 쿠키의 보관 기간 명시
- [ ] **접근 가능성**: 쿠키 설정에 쉽게 접근 가능 (footer 링크 등)

### GDPR 준수 요구사항

- [ ] 쿠키 사용 전 명시적 동의 (Pre-consent)
- [ ] "모두 거부" 옵션이 "모두 동의"만큼 쉽게 접근 가능
- [ ] 세분화된 동의 옵션 (쿠키 카테고리별)
- [ ] 동의 기록 보관 (언제, 무엇에 동의했는지)
- [ ] 동의 철회 메커니즘
- [ ] 명확하고 평이한 언어 사용

### 한국 개인정보보호법 준수

- [ ] 쿠키 정책을 개인정보 처리방침에 포함
- [ ] 사용자 선택권 보장
- [ ] 쿠키를 통한 개인정보 수집 시 사전 고지
- [ ] 행태정보 수집 시 별도 동의
- [ ] 만 14세 미만 아동의 경우 법정대리인 동의

### 기술적 구현 체크리스트

- [ ] 동의 전까지 비필수 쿠키 차단
- [ ] 동의 상태를 localStorage/쿠키에 저장
- [ ] 동의 상태에 따라 스크립트 로딩 제어
- [ ] 쿠키 배너가 모든 페이지에서 표시
- [ ] 모바일 반응형 디자인
- [ ] 접근성 준수 (ARIA 레이블, 키보드 내비게이션)
- [ ] 페이지 로딩 속도에 영향 최소화

### 문서화 요구사항

- [ ] **쿠키 정책 페이지**: 상세한 쿠키 사용 설명
- [ ] **쿠키 목록**: 사용하는 모든 쿠키의 이름, 목적, 기간
- [ ] **업데이트 이력**: 쿠키 정책 변경 시 버전 관리
- [ ] **제3자 목록**: 데이터를 공유하는 제3자 명시

### 테스트 체크리스트

- [ ] 다양한 브라우저에서 테스트 (Chrome, Safari, Firefox, Edge)
- [ ] 모바일 기기에서 테스트
- [ ] 쿠키 차단 시나리오 테스트
- [ ] 동의 철회 후 쿠키 삭제 확인
- [ ] 필수 쿠키만 허용 시 사이트 기능 확인

## 쿠키 종류별 설명 템플릿

### 필수 쿠키 (Strictly Necessary Cookies)

```
이름: session_id
목적: 사용자 세션 유지 및 로그인 상태 관리
기간: 세션 종료 시 (브라우저 닫기)
제공자: MUIN
```

### 분석 쿠키 (Analytics Cookies)

```
이름: _ga, _gid
목적: Google Analytics를 통한 사이트 사용 패턴 분석
기간: _ga (2년), _gid (24시간)
제공자: Google LLC
개인정보 처리: IP 주소 익명화 처리
```

### 마케팅 쿠키 (Marketing Cookies)

```
이름: _fbp
목적: Facebook 픽셀을 통한 광고 효과 측정 및 맞춤형 광고
기간: 3개월
제공자: Meta Platforms, Inc.
```

### 기능 쿠키 (Functional Cookies)

```
이름: language, theme
목적: 사용자 선호 언어 및 테마 설정 저장
기간: 1년
제공자: MUIN
```

## 구현 참고사항

### 권장 쿠키 동의 라이브러리

1. **Cookiebot** - GDPR/ePrivacy 완벽 준수
2. **OneTrust** - 엔터프라이즈급 솔루션
3. **Cookie Consent by Osano** - 오픈소스, 커스터마이징 용이
4. **GDPR Cookie Consent** - WordPress 플러그인

### 자체 구현 시 고려사항

```javascript
// 동의 전까지 스크립트 차단 예제
<script type="text/plain" data-cookie-consent="analytics">
  // Google Analytics 코드
</script>

// 동의 후 활성화
if (cookieConsent.analytics) {
  activateScript('analytics');
}
```

[🍪 쿠키 설정] [개인정보 처리방침] [이용약관]
