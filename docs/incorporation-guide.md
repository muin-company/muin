## 🛠 설립 방법 비교

### 1. 직접 설립 (DIY)

**비용:** 약 20만원 (필수 비용만)
**소요시간:** 3-7일
**난이도:** ★★★★☆

**절차:**
1. 정관 작성
2. 발기인 총회 (1인이면 발기인 결정서)
3. 자본금 납입 (은행)
4. 설립등기 신청 (등기소)
5. 사업자등록 (세무서/홈택스)

**장점:** 가장 저렴
**단점:** 서류 작성 복잡, 실수 리스크

### 2. 온라인 플랫폼

**비용:** 20-40만원 + 필수비용
**소요시간:** 1-3일
**난이도:** ★★☆☆☆

**주요 서비스:**
- 법인설립 원스톱 서비스 (정부)
- 스타트업 전문 법무 플랫폼
- 세무사무소 온라인 서비스

**장점:** 편리, 실수 최소화
**단점:** 수수료 발생

### 3. 법무사/세무사 대행

**비용:** 30-50만원 + 필수비용
**소요시간:** 3-5일
**난이도:** ★☆☆☆☆

**장점:** 전문가가 처리, 안전
**단점:** 비용 높음

## ✅ 권장 방안

### 옵션 A: 최저 비용 (직접)

**총 비용:** ~20만원
**조건:** 시간 여유 있고, 서류 작업 가능할 때

**필요 서류:**
- 정관 (인터넷 양식 활용)
- 발기인 결정서
- 주금납입증명서 (은행 발급)
- 취임승낙서
- 인감신고서
- 등기신청서

### 옵션 B: 가성비 (온라인 플랫폼) ⭐ 권장

**총 비용:** 40-50만원
**장점:** 
- 실수 방지
- 시간 절약
- 사후 지원 (정관 변경 등)

**추천 서비스:**
- 정부 법인설립 원스톱 서비스 (startbiz.go.kr)
- 헬프미 법률사무소 (온라인)
- 세무회계사무소 온라인 패키지

### 옵션 C: 안전 최우선 (전문가)

**총 비용:** 50-70만원
**적합:** 복잡한 지분 구조, 투자 계획 있을 때

## 📝 1인 주식회사 특이사항

1. **발기인 = 주주 = 대표이사** 가능
2. 이사회 구성 불필요 (1인 이사 가능)
3. 감사 선임 의무 없음 (자본금 10억 미만)
4. 주주총회 = 1인 결정으로 대체 가능

---

## ⚠️ 주의사항

1. **자본금 납입**: 실제 납입 필요 (가장납입 불법)
2. **사업장 주소**: 자택 가능, 단 임대차계약서 필요할 수 있음
3. **업종 선정**: 향후 확장 고려해서 넓게 설정
4. **4대보험**: 대표이사도 가입 의무 (건강보험 등)
5. **세금**: 법인세, 부가가치세, 원천세 등 신고 의무
