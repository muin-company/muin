## 💰 Mac Studio vs API 분석

### 결론: 현 단계에서는 API 유지 권장

| 비교 항목 | Mac Studio | Claude API |
|----------|------------|------------|
| 초기 비용 | 1,000만원 | 0원 |
| 월 비용 (활발) | 전기 ~5만원 | ~22만원 |
| 성능 | Llama 70B 수준 | Opus 4.5 (최고) |
| 손익분기 | - | 3.8년 |

### 권장사항
- **지금**: API 유지 (최고 성능, 사업 검증 우선)
- **나중**: 월 50만원+ 초과시 재검토

**상세 분석:** `/muin/docs/mac-studio-vs-api-analysis.md`
