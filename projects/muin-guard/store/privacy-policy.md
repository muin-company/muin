# MUIN Guard 개인정보처리방침

**최종 수정일:** 2026-02-02

## 1. 개요

MUIN Guard("본 서비스")는 사용자의 프라이버시를 최우선으로 생각합니다. 본 개인정보처리방침은 MUIN Guard 크롬 확장 프로그램의 데이터 수집 및 처리 방식을 설명합니다.

## 2. 핵심 원칙

**우리는 사용자의 데이터를 수집하지 않습니다.**

MUIN Guard는 100% 로컬에서 작동하며, 모든 데이터는 사용자의 브라우저에만 저장됩니다. 어떠한 데이터도 외부 서버로 전송되지 않습니다.

## 3. 저장되는 데이터

다음 데이터가 **사용자의 브라우저 로컬 스토리지**에만 저장됩니다:

### 3.1 대화 기록
- AI 대화 내용 (처음 500자)
- 플랫폼 정보 (ChatGPT, Claude 등)
- 타임스탬프
- 탐지된 위험 정보

### 3.2 알림
- 위험 탐지 알림 내역
- 위험 유형 및 심각도

### 3.3 설정
- 모니터링 활성화 여부
- 알림 설정
- 데이터 보관 기간

## 4. 데이터 전송

MUIN Guard는 **어떠한 데이터도 외부로 전송하지 않습니다.**

- 분석 데이터 전송: ❌
- 사용 통계 전송: ❌
- 광고 목적 데이터 전송: ❌
- 제3자 데이터 공유: ❌

## 5. AI 분석 기능 (선택)

WebGPU AI 분석 기능을 활성화하면:

- AI 모델이 **사용자의 기기에서 직접** 실행됩니다
- 모델 파일은 CDN에서 다운로드 후 로컬에 캐싱됩니다
- 분석 과정에서 어떠한 데이터도 서버로 전송되지 않습니다

## 6. 데이터 삭제

사용자는 언제든지 저장된 모든 데이터를 삭제할 수 있습니다:

1. MUIN Guard 대시보드 열기
2. 설정 → "모든 데이터 삭제" 클릭

또는 크롬 확장 프로그램을 제거하면 모든 데이터가 자동으로 삭제됩니다.

## 7. 권한 사용 목적

| 권한 | 목적 |
|------|------|
| storage | 로컬에 설정 및 기록 저장 |
| notifications | 위험 탐지 알림 표시 |
| offscreen | WebGPU AI 분석 실행 |
| host_permissions | AI 사이트 대화 모니터링 |

## 8. 아동 개인정보

MUIN Guard는 13세 미만 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다.

## 9. 변경 사항

본 개인정보처리방침이 변경될 경우, 확장 프로그램 업데이트 및 스토어 페이지를 통해 공지합니다.
