## 간단한 설명 (132자 이내)
AI 대화에서 개인정보, API 키, 위험 명령어를 자동 탐지. 100% 로컬 저장으로 프라이버시 보장. WebGPU AI 분석 지원.

## 상세 설명

### 🛡️ MUIN Guard - AI 대화를 안전하게

AI와 대화할 때 실수로 민감한 정보를 노출하고 있지 않나요?

MUIN Guard는 ChatGPT, Claude 등 AI 서비스 사용 시 **실시간으로 위험을 탐지**하고 알려주는 보안 확장 프로그램입니다.

---

### ✨ 주요 기능

**🔍 실시간 위험 탐지**
- 이메일, 전화번호, 신용카드 번호 등 개인정보
- API 키, 비밀번호, 토큰 등 민감 정보
- rm -rf, DROP TABLE 등 위험 명령어
- SQL 인젝션, XSS 등 보안 위협 패턴

**🔒 100% 프라이버시 보장**
- 모든 데이터는 브라우저 로컬에만 저장
- 서버 전송 없음 - 우리도 당신의 대화를 볼 수 없습니다
- 원할 때 언제든 데이터 삭제 가능

**🧠 AI 분석 (선택 기능)**
- WebGPU를 활용한 로컬 LLM 분석
- Llama Guard 3 기반 고급 위험 탐지
- 추가 비용 없이 무료 제공

**📊 대시보드**
- 대화 기록 및 알림 관리
- 플랫폼별 사용 현황
- 상세 설정 옵션
