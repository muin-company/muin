# SEO Meta Tags Templates for MUIN Products

Each product page should include:
- HTML `<title>` and `<meta>` tags
- Open Graph (OG) tags for social sharing
- Twitter Card tags

## Additional Meta Tags (Recommended for All Pages)

```html
<!-- Viewport & Mobile -->
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<meta name="theme-color" content="#000000" />

<!-- Favicon & App Icons -->
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png" />
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png" />
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
<link rel="manifest" href="/site.webmanifest" />

<!-- Security & Privacy -->
<meta name="robots" content="index, follow" />
<meta name="googlebot" content="index, follow" />
<meta http-equiv="X-UA-Compatible" content="IE=edge" />

<!-- Canonical URL (prevents duplicate content) -->
<link rel="canonical" href="https://example.com/current-page" />

<!-- Language Alternates -->
<link rel="alternate" hreflang="ko" href="https://example.com/" />
<link rel="alternate" hreflang="en" href="https://example.com/en/" />
<link rel="alternate" hreflang="x-default" href="https://example.com/" />
```

## OG Image Guidelines

**Dimensions:** 1200×630px (1.91:1 ratio)  
**Format:** PNG or JPG  
**Max Size:** < 1MB for faster loading  

**Content:**
- Product logo + tagline
- Clean, readable text (min 40px font size)
- Brand colors
- No clutter — simple is better

**File naming:**
- `/og-image.png` (Korean default)
- `/og-image-en.png` (English)
- Product-specific: `/og-image-gumsi.png`

## Dynamic Meta Tag Template (Next.js/React)

```jsx
import Head from 'next/head';

export default function SEOHead({ 
  title, 
  description, 
  image = '/og-image.png',
  url,
  locale = 'ko_KR'
}) {
  const fullUrl = `https://muin.company${url}`;
  const fullImage = `https://muin.company${image}`;
  
  return (
    <Head>
      <title>{title}</title>
      <meta name="title" content={title} />
      <meta name="description" content={description} />
      
      <meta property="og:type" content="website" />
      <meta property="og:url" content={fullUrl} />
      <meta property="og:title" content={title} />
      <meta property="og:description" content={description} />
      <meta property="og:image" content={fullImage} />
      <meta property="og:locale" content={locale} />
      
      <meta property="twitter:card" content="summary_large_image" />
      <meta property="twitter:url" content={fullUrl} />
      <meta property="twitter:title" content={title} />
      <meta property="twitter:description" content={description} />
      <meta property="twitter:image" content={fullImage} />
      
      <link rel="canonical" href={fullUrl} />
    </Head>
  );
}
```
