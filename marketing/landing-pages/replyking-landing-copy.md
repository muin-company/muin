## Features Section

### 💬 Feature 2: 자연스러운 대화
**AI인지 모를 정도로 자연스럽게**
"재입고 언제 돼요?", "배송비 얼마예요?" - 실제 사람처럼 자연스럽게 답변하고 이모티콘도 사용합니다.

### 📦 Feature 3: 상품 정보 자동 안내
**재고/가격/옵션 실시간 연동**
쇼핑몰과 연동해서 상품 정보를 자동으로 안내하고, 품절 시 대체 상품을 추천합니다.

### 💳 Feature 4: 결제 링크 자동 발송
**문의에서 결제까지 한 번에**
구매 의사가 확인되면 자동으로 결제 링크를 발송. DM에서 바로 구매 전환됩니다.

### 📊 Feature 5: 실시간 대시보드
**모든 대화가 한눈에**
답변률, 전환율, 매출 통계를 실시간으로 확인하고, AI가 답변 못한 건은 알림으로 받습니다.

### 🎯 Feature 6: 맞춤형 학습
**당신의 브랜드 톤앤매너로**
상품 정보, 자주 묻는 질문, 말투까지 학습시켜서 브랜드에 딱 맞는 답변을 만듭니다.

## How It Works

### Step 1: 인스타 계정 연결
간단한 인증만으로 3분 안에 연결 완료. 별도 앱 설치 불필요.

### Step 2: AI 학습
상품 목록, FAQ, 브랜드 가이드를 입력하면 AI가 자동으로 학습합니다.

### Step 3: 자동 운영 시작
이제 DM은 AI가 알아서 답변. 복잡한 문의만 당신에게 알림이 갑니다.

## FAQ

### Q1: 인스타그램 계정이 차단되지 않나요?
공식 Instagram Graph API를 사용하며, Meta의 정책을 100% 준수합니다. 수천 개 계정이 안전하게 사용 중입니다.

### Q2: 설정이 어렵지 않나요?
3분이면 연결 완료됩니다. 복잡한 코딩이나 기술 지식은 전혀 필요 없습니다.

### Q3: AI가 이상한 답변을 하면 어떡하죠?
모든 대화를 실시간 모니터링할 수 있고, AI가 확신 없는 답변은 자동으로 당신에게 넘깁니다. 언제든 수동 개입 가능합니다.

### Q4: 어떤 쇼핑몰과 연동되나요?
스마트스토어, 카페24, 아임웹, 고도몰 등 주요 쇼핑몰 모두 지원하며, 수동 입력도 가능합니다.

### Q5: 결제는 어떻게 처리되나요?
결제 링크를 자동으로 발송하고, 실제 결제는 기존 쇼핑몰이나 토스/네이버페이로 진행됩니다.

### Q6: 여러 계정을 동시에 운영할 수 있나요?
프로 플랜부터 여러 계정을 하나의 대시보드에서 관리할 수 있습니다.

### Q7: 카카오톡이나 네이버 톡톡도 지원하나요?
비즈니스 플랜부터 멀티 채널 통합을 지원합니다. 하나의 AI로 모든 채널 관리 가능합니다.

### Q8: 해외 고객 문의도 처리 가능한가요?
영어, 중국어, 일본어 등 30개 언어를 지원하며, 자동 번역 기능도 제공합니다.

### Q9: 중간에 플랜 변경 가능한가요?
언제든지 업그레이드/다운그레이드 가능하며, 남은 기간은 일할 계산됩니다.

### Q10: 해지는 어떻게 하나요?
언제든 클릭 한 번으로 해지 가능하며, 위약금이나 해지 수수료는 없습니다.
