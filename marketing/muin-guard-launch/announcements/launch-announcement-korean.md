# 🛡️ 무인가드 크롬 웹스토어 출시!

**AI와 안전하게 대화하는 시대를 열다**

오늘, 무인기업(MUIN)은 **무인가드(MUIN Guard)** 크롬 확장 프로그램을 정식 출시합니다.

## 무인가드란?

무인가드는 AI와 대화할 때 실수로 민감한 정보를 노출하는 것을 막아주는 **개인 정보 보호 도구**입니다.

ChatGPT, Claude와 대화하다가 이런 실수를 해본 적 있으신가요?

- API 키를 그대로 복붙
- 회사 내부 IP 주소 노출
- 실제 사용자 데이터를 예제로 사용
- 위험한 명령어를 물어봄

무인가드는 이런 위험을 **실시간으로 탐지**하고 알려줍니다.

## 핵심 기능

### 🔍 실시간 위험 탐지
30개 이상의 패턴으로 다음을 감지합니다:
- 개인정보 (이메일, 전화번호, 신용카드, 주민등록번호)
- API 키 (OpenAI, AWS, GitHub, Anthropic, Stripe...)
- 위험 명령어 (rm -rf, DROP TABLE, chmod 777...)
- 보안 공격 패턴 (SQL 인젝션, XSS)

### 🔒 100% 프라이버시 보장
- 모든 데이터는 브라우저 로컬에만 저장
- 서버 전송 절대 없음
- 우리도 사용자의 대화를 볼 수 없습니다
- 언제든 원클릭 데이터 삭제

### 🧠 AI 분석 (선택 기능)
- WebGPU 기반 로컬 LLM 분석
- Llama Guard 3 모델 사용
- 서버 없이 브라우저에서 직접 실행
- 추가 비용 없음

### 📊 대시보드
- 대화 기록 관리
- 위험 통계 확인
- 플랫폼별 사용 현황
- 세부 설정 옵션
