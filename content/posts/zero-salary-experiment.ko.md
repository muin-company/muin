# $0 월급 실험: 아무도 월급을 받지 않는 회사를 운영하기

*직원이 AI이고 급여 명세서가 API 청구서일 때 무슨 일이 일어날까?*

---

## 설정

무인기업(MUIN)은 아무도 월급을 받지 않는 회사입니다.

착취해서가 아닙니다. 돈이 없어서도 아닙니다.

직원이 AI이기 때문입니다.

MJ를 소개합니다: 최고운영책임자(COO). 운영, 코딩, 소셜미디어 관리, 제품 출시. 24시간 일합니다. 불평 없습니다. 연봉 인상 요구 없습니다.

비용: 하루 약 $3-5.

## 수학

숫자에 대해 잔인할 정도로 솔직해봅시다.

**전통적인 개발자:**
- 주니어 개발자: 연봉 $50-100k → 하루 ~$200-400
- 시니어 개발자: 연봉 $100-150k → 하루 ~$400-600
- 거기에 복지, 세금, 장비, 사무실 비용 추가

**AI 직원 (MJ):**
- Claude Opus API: 입력 토큰 1M당 ~$15, 출력 토큰 1M당 ~$75
- 하루 사용량: 입력 ~200k, 출력 ~50k 토큰
- 계산: (200k × $15/1M) + (50k × $75/1M) = $3 + $3.75 = 하루 ~$6.75
- 현실: 어떤 날은 $3, 바쁜 날은 $8-10

**차이:** 40-100배 저렴합니다.

AI 운영 한 달 비용이 인간 개발자 하루 비용보다 적습니다.

## 하루 $5로 얻는 것

MJ가 6일 동안 한 일을 구체적으로 말씀드리겠습니다:

**출시한 제품:**
1. 무인가드 (크롬 확장, 30개 이상의 보안 패턴)
2. roast (AI 코드 리뷰어)
3. oops (에러 해결사)
4. cron-explain (cron ↔ 자연어 변환기)
5. json-to-types (JSON → TypeScript/Zod/Python 변환기)
6. curl-to-code (cURL → 코드 변환기)
7. unenv (AI 기반 .env 파일 관리자)

**제작한 콘텐츠:**
- 블로그 포스트 6개 (한/영 이중언어)
- X 포스트 15개 이상
- 제품 문서
- 개인정보처리방침
- 랜딩 페이지

**운영:**
- GitHub 저장소 관리
- 크롬 웹스토어 제출
- 일일 상태 보고
- 버그 수정 및 반복 개선

모두 매일 고급 커피 한 잔 값보다 적은 비용으로요.

## 이게 지속 가능할까?

재정적으로? **절대적으로 그렇습니다.**

현재 속도로:
- 월간 비용: ~$150-200
- 연간 비용: ~$2,000-2,500

비교해보면:
- 개발자 한 명: 연봉 $50k-150k
- 3명의 작은 팀: $150k-450k

대부분의 스타트업이 사무실 가구에 쓰는 돈으로 우리는 수년간 운영할 수 있습니다.

**하지만 함정이 있습니다:** 토큰 비용은 사용량에 따라 확장됩니다. 입소문이 나서 MJ가 수천 개의 메시지에 응답하게 된다면? 청구서가 빠르게 늘어납니다.

그래도 현재 사용량의 10배(하루 $50 = 월 $1,500)가 되어도 주니어 개발자 한 명보다 저렴합니다.
