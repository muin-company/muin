# MUIN Guard 개인정보처리방침

**최종 수정일: 2026년 2월 4일**

MUIN(무인기업)은 사용자의 개인정보를 중요하게 생각합니다. 본 개인정보처리방침은 MUIN Guard Chrome 확장 프로그램("서비스")이 사용자 정보를 어떻게 수집, 사용, 보호하는지 설명합니다.

## 1. 수집하는 정보

### 1.1 자동 수집 정보
- **대화 내용 분석 데이터**: ChatGPT, Claude 등 AI 대화 서비스에서 사용자가 입력하는 텍스트를 분석합니다. 이 데이터는 민감한 정보(개인정보, API 키, 비밀번호 등) 탐지 목적으로만 사용됩니다.
- **탐지 기록**: 위험으로 탐지된 패턴의 유형과 시간 정보를 저장합니다.
- **설정 정보**: 사용자의 확장 프로그램 설정을 저장합니다.

### 1.2 수집하지 않는 정보
- 사용자의 실제 대화 내용을 외부 서버로 전송하지 않습니다.
- 로그인 정보, 결제 정보 등을 수집하지 않습니다.

## 2. 정보의 저장

**모든 데이터는 사용자의 브라우저 로컬 저장소에만 저장됩니다.**

- Chrome의 `chrome.storage.local` API를 사용합니다.
- 데이터는 사용자의 기기에서 벗어나지 않습니다.
- MUIN 또는 제3자 서버로 데이터가 전송되지 않습니다.

## 3. 정보의 사용

수집된 정보는 다음 목적으로만 사용됩니다:
- 민감한 정보의 실시간 탐지 및 알림
- 대시보드에서 탐지 기록 표시
- 사용자 설정 유지

## 4. 정보의 공유

**MUIN Guard는 사용자 데이터를 제3자와 공유하지 않습니다.**

- 광고 네트워크에 데이터를 판매하거나 전송하지 않습니다.
- 분석 서비스에 데이터를 전송하지 않습니다.
- 어떠한 외부 서버와도 통신하지 않습니다.

## 5. 데이터 삭제

사용자는 언제든지 저장된 데이터를 삭제할 수 있습니다:
1. 확장 프로그램 대시보드에서 "데이터 삭제" 버튼 클릭
2. Chrome 설정에서 확장 프로그램 제거

확장 프로그램 제거 시 모든 로컬 데이터가 자동으로 삭제됩니다.

## 6. 보안

- 모든 데이터는 사용자 기기 내에서만 처리됩니다.
- 네트워크 통신이 발생하지 않아 데이터 유출 위험이 없습니다.
- AI 분석 기능(WebGPU LLM)도 완전히 로컬에서 실행됩니다.

## 7. 권한 설명

MUIN Guard는 다음 권한을 요청합니다:

| 권한 | 사용 목적 |
|------|----------|
| `storage` | 로컬에 대화 기록 및 설정 저장 |
| `notifications` | 위험 탐지 시 알림 표시 |
| `offscreen` | WebGPU AI 분석 기능 |
| 호스트 권한 | ChatGPT, Claude 사이트 접근 |

## 8. 변경 사항

본 개인정보처리방침이 변경될 경우, 확장 프로그램 업데이트 노트와 본 페이지를 통해 안내합니다.
